# AWS Lambda para salvar imagem e arquivos em um bucket S3 para .Net 8

Projeto Lambda Function simples em .net 8 onde é passando um inteiro para a requição para direcionar a invocação do serviço
Install Amazon.Lambda.Tools Global Tools if not already installed.


Fazer o deploy no AWS Lamabda
```
    cd "AWSS3Service/src/AWSS3Service"
    dotnet lambda deploy-function
```
